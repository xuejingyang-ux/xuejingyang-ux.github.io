# Jingyang Xue Academic Homepage

A clean bilingual academic homepage for GitHub Pages.

This site is a pure static project built with HTML, CSS, and a small amount of JavaScript. It is designed for easy editing, direct deployment on GitHub Pages, and reuse as an open-source personal academic homepage template.

## Preview

Open `index.html` through a local static server or deploy the repository with GitHub Pages.

## Customize

- Edit `index.html` to update profile, news, research, publications, projects, honors, and contact information.
- Edit `static/css/main.css` to adjust colors, layout, spacing, and responsive styles.
- Replace `static/assets/img/photo.png` with your own square profile photo.
- Replace `static/assets/favicon.ico` if you want a custom browser icon.

## Deploy

Create a repository named:

```text
xuejingyang-ux.github.io
```

Then push this project to the `main` branch and enable GitHub Pages from `Settings -> Pages -> Deploy from a branch -> main -> / (root)`.
